# OpenWebUI Configuration

This directory stores OpenWebUI settings that can be version controlled.

## Exporting Settings

1. Go to OpenWebUI Admin Panel
2. Settings → Export
3. Save to this directory
4. Commit to git (sanitize any secrets first!)

## Importing Settings

1. Start fresh OpenWebUI instance
2. Admin Panel → Settings → Import
3. Select file from this directory

## What to Store Here

- Model configurations
- System prompts
- UI customizations
- Function metadata (auto-generated by scripts)

## What NOT to Store

- API keys (use .env)
- User data
- Chat histories
